What is this?

This widget lets you play around with different tonics and modes to see what the resulting diatonic scale looks like. I only made it because the functionality was required in order to insert scale degrees given different chord symbols - and because I found it interesting.

A personal rambling

If you've ever learned to read music, you might know that all of the common modes will follow the same key signature flat/sharp pattern (BEADGCF and FCGDAEB) because they are all an offset of Ionion mode. So what happens when you re-order the whole/half step pattern? For example, try adding the pattern 2,1,2,1,2,2,2 and name it "Panda". You'll notice Panda mode no longer follows the flat/sharp pattern. This is just one way to create some funky sounding scales when compared to the traditional Ionian/Aeolian sounds we're trained to hear. There are plenty more ways to tamper with the fundamentals in western music and create something blasphemous.
